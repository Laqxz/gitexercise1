# gitexercise1
An exercise for clone, commit and pullrequest.

# Steps to perform
 - clone the repository locally
 - create a new branch
 - fix all errors and add missing operators
 - commit the changes locally
 - create pull request locally and upload it in moodle

# Commands needed
	git status
	git clone
	git checkout -b
	git add
	git commit
  	git request-pull
Compile and test:

	gcc calculator.c
	./a.out
