# gitexercise1
An exercise for clone, commit and pullrequest.

# Steps to perform
 - clone the repository locally
 - create a new branch
 - fix all errors and add missing operators
 - create pull request locally and upload it in moodle

# Commands needed 
	git clone
	git commit
  	git request-pull
Compile and test:

	gcc calculator.c
	./a.out
